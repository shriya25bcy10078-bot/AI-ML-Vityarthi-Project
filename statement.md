PROBLEM :- 
Individuals and households struggle with effective daily expense tracking and financial management. The existing manual methods of recording expenses in notebooks or scattered digital notes lead to poor financial visibility, inefficient categorization, lack of budgeting and no historical analysis.

SOLUTION :- 
My project builds a basic expense tracking system, through which user can categorize there expenses into categories. 

SCOPE OF PROJECT :- 
This project is single user-friendly application. It takes several input from user. Inputs include name, amount, category and description.

TARGET USER :- 
This application can be used by all age group people to maintain record of there daily expenses. 
Primary target users are students and young corporate employees. And secondary target user can be single household people. 
Students and young adults can track pocket money, monitoring spending on food, transportation and entertainment. 
Young Corporate Employees can manage there monthly salaries, rents, utilities and other essential daily expense. 
Individual Household People can track there groceries, manage multiple financial responsibilities and build emergency funds.

HIGH-LEVEL FEATURES :- 
My Project contains several features.
Adds expense - Creates new expense records with amount, category, description and optional tags.
List Expense -  Expenses are in sortable list with key columns.
Find Expense - Expenses can be found from the sorted list
Update Expense - Edit existing expense entries and persist changes.
Delete Expense - Remove single or multiple expense records with confirmation and optional undo.
Total Expense - Total expense is displayed
These features are intentionally high-level; each can be broken down into specific user stories or tasks during implementation.
